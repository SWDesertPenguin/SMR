    SMR — Standalone Markdown Reader

    Open one or more .md files as tabs, edit on the left, and watch the live preview on the right. Drag the dividers to resize any pane; in split view, scrolling one side scrolls the other.

    Getting started

    • Open a file with the 📂 button, the File menu, or by dropping a .md file onto the window.
    • Start a blank file with 🖉, then Save to give it a name.
    • Use ▥ to cycle between split, preview-only, and editor-only views.
    • Click a heading in Contents to jump to it; the list tracks your scroll position.
